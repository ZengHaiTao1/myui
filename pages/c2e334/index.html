# base-selectList 自定义数据下拉组件

根据传入的参数、url、选项字段生成选择器。

# 基本用法

适用于普通多选下拉框

    无数据 

    <template>
  <base-selectList
    apiUrl="/api/dataLoad/list"
    v-model="value"
    @getSelectList="getUserTypeList"
    :propsOption="{ label: 'name', value: 'code' }"
  ></base-selectList>
</template>
<script>
  export default {
    name: "allSelect",
    data() {
      return {
        value: ""
      };
    },
    methods: {
      getUserTypeList(value, option) {
        this.value = value;
        console.log(value, option);
      }
    }
  };
</script>

    # 属性

    参数 说明 类型 默认值
    apiUrl 获取数据 url String ''
    propsOption 定义下拉选项的值、标签及返回数据的获取位置 Object { label: "name", value: "value", result: "data.data",}
    params 获取数据的参数 Object {}
    isBase 是否在国网版本调用请求 Boolean false

    # 事件

    名称 说明 参数 默认值
    getSelectList 接口数据获取后触发 选中的值、所有数据列表 --
