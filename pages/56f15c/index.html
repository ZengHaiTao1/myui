# base-timeSelect 时间级联组件

用于时间区间查询

# 基本用法

封装开始时间不可以超过结束时间功能

-

Smodel.sync Smodel.sync分别对应开始时间和结束时间的双向绑定参数，数组形式。

<template>
  <div>
    <el-form :inline="true" :model="forecastSetForm">
      <base-timeSelect
        :Smodel.sync="startTime"
        :Emodel.sync="endTime"
      ></base-timeSelect>
    </el-form>
  </div>
</template>
<script>
  export default {
    data() {
      return {
        forecastSetForm: {},
        startTime: "00:15",
        endTime: "00:15",
      };
    },
    mounted() {},

    methods: {},
  };
</script>
