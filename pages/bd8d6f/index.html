# base-allSelect 全选下级下拉组件

全选、全部取消下拉选项功能。

# 基本用法

适用于普通多选下拉框

  • 全选
  • label1
  • label2

无数据 

<template>
  <base-allSelect v-model="value" :option="options" size="small">
  </base-allSelect>
</template>
    <script>
      export default {
        name: "allSelect",
        data() {
          return {
            value: "1",
            options: [
              { name: "name1", id: "1" },
              { name: "name2", id: "2" }
            ]
          };
        },
      };
    </script></template
  ></template
>

# 自定义选项字段

  • 全选
  • name1
  • name2

无数据

通过listOption字段定义下拉框中选项的值和标签

<template>
  <base-allSelect
    v-model="value"
    :option="options"
    :listOption="{ label: 'name', value: 'id' }"
    size="small"
  ></base-allSelect>
</template>
<script>
  export default {
    name: "allSelect",
    data() {
      return {
        value: ["2"],
        options: [
          { name: "name1", id: "1" },
          { name: "name2", id: "2" }
        ]
      };
    }
  };
</script>

# 联级选择器

  • 全选
  • 一级1
  • 二级1
  • 三级1
  • 三级2
  • 二级2
  • 一级2

无数据

通过isChild配置是否是多级，listOption.child字段定义下拉选项中的层级

<template>
  <base-allSelect
    v-model="value"
    :option="options"
    isChild
    :listOption="{ label: 'name', value: 'id', child: 'child' }"
    size="small"
  ></base-allSelect>
</template>
<script>
  export default {
    name: "allSelect",
    data() {
      return {
        value: ["1"],
        options: [
          {
            name: "一级1",
            id: "1",
            child: 0
          },
          {
            name: "二级1",
            id: "1-1",
            child: 1
          },
          {
            name: "三级1",
            id: "1-1-1",
            child: 2
          },
          {
            name: "三级2",
            id: "1-1-2",
            child: 2
          },
          { name: "二级2", id: "1-2", child: 1 },
          { name: "一级2", id: "2", child: 0 }
        ]
      };
    }
  };
</script>

# 属性

参数 说明 类型 默认值
option 下拉选项列表 array []
listOption 定义下拉框中选项的值和标签 Object {}
isChild 选项是否是多级 Boolean false

# 事件

名称 说明 参数 默认值
input 选项改变、点击全选时触发 选中的值 --
changes 选项改变时触发 选中的值 --

# 方法

名称 说明 参数
changeSelect 使选项改改变 选中的值
selectAll 使全选状态取反 --
